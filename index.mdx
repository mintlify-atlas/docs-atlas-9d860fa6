<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-3">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with Mas Agua in minutes</p>
  
  <Steps>
    <Step title="Access the platform">
      Log in to your Mas Agua instance using your credentials. For desktop users, download and install the Tauri-based application for native performance.
      
      ```bash
      # Download the latest desktop release
      https://github.com/CspmIT/mas-agua-front/releases
      ```
    </Step>
    
    <Step title="Configure your data sources">
      Connect your PLC devices and configure InfluxDB variables to start receiving real-time data from your water infrastructure.
      
      Navigate to **Configuration > Variables** to set up your data sources and define measurement parameters.
    </Step>
    
    <Step title="Create your first dashboard">
      Build a custom dashboard with live monitoring widgets. Choose from gauge charts, liquid fill indicators, boolean status displays, and interactive maps.
      
      <Note>
        Dashboards automatically refresh every 30 seconds to display the latest data from your sensors and devices.
      </Note>
    </Step>
    
    <Step title="Set up alarms">
      Configure alarm thresholds to receive notifications when critical parameters exceed safe operating ranges.
      
      Go to **Configuration > Alarms** to define your monitoring rules and notification preferences.
    </Step>
  </Steps>
</div>
